# Changelog

All notable changes follow [Keep a Changelog](https://keepachangelog.com/en/1.1.0/) and semantic versioning.

## [Unreleased]

## [0.2.0] - 2026-07-15

### Added

- Full English and Simplified Chinese README guides covering features, installation, usage, integrations, and privacy.
- Separate English and Simplified Chinese SVG banners.
- A 12-second GSAP product tour demonstrating search, signal loading, and local installation with synthetic data.

### Changed

- Renamed the product, plugin ID, package, installation directory, and repository to **Toworkboard**.
- Renamed the original geometric mark to **Canopy T** while preserving the forest visual system.
- Added migration paths for the previous secure-storage and macOS Keychain identifiers.

## [0.1.0] - 2026-07-15

### Added

- Local full-text note search and frequent-note history.
- Local task list and vault activity summaries.
- Read-only Gmail metadata and QQ Mail IMAP views.
- GitHub star radar and Hacker News signals.
- Independent fixed-height Mailroom panes.
- GSAP Canopy signal loading motion with reduced-motion support.
- Secure credential storage and legacy GitHub token migration.
- Original Canopy T and Canopy rail identity by Totoro.
- Synthetic SVG, PNG, and GIF documentation assets.
- Privacy checks, tests, linting, build verification, and secret scanning.
