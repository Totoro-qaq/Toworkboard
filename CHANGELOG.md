## [0.1.0] - 2026-07-15

### Added

- Local full-text note search and frequent-note history.
- Local task list and vault activity summaries.
- Read-only Gmail metadata and QQ Mail IMAP views.
- GitHub star radar and Hacker News signals.
- Independent fixed-height Mailroom panes.
- GSAP Canopy signal loading motion with reduced-motion support.
- Secure credential storage and legacy GitHub token migration.
- Original Forest T and Canopy rail identity by Totoro.
- Synthetic SVG, PNG, and GIF documentation assets.
- Privacy checks, tests, linting, build verification, and secret scanning.
