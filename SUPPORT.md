# Support

Toworkboard is an early public preview maintained in spare time.

- Use the bug form for a reproducible defect.
- Use the feature form for a focused proposal.
- Use GitHub Discussions for setup questions once discussions are enabled.
- Use private vulnerability reporting for security or privacy problems.

Before posting, run `npm run verify`, check the latest release notes, and reproduce with synthetic data. Never publish OAuth tokens, authorization codes, account passwords, personal email addresses, real note titles, vault paths, or mailbox screenshots.
